# Nihat Aliyev | Tüm Bağlantılar

Modern ve profesyonel bir kişisel bağlantı sayfası. Tüm sosyal medya hesaplarımı ve önemli bağlantılarımı tek bir yerde topluyorum.

## 🌟 Özellikler

- **Modern Tasarım**: Temiz ve profesyonel görünüm
- **Responsive**: Tüm cihazlarda mükemmel görünüm
- **Animasyonlar**: Yumuşak geçişler ve hover efektleri
- **Erişilebilirlik**: WCAG standartlarına uygun
- **SEO Optimized**: Arama motorları için optimize edilmiş
- **Hızlı Yükleme**: Performans odaklı tasarım
- **PWA Ready**: Progressive Web App desteği

## 🚀 Teknolojiler

- **HTML5**: Semantic markup
- **CSS3**: Modern CSS özellikleri (Grid, Flexbox, Custom Properties)
- **JavaScript**: Vanilla JS ile interaktif özellikler
- **Font Awesome**: İkon kütüphanesi
- **Google Fonts**: Inter font ailesi

## 🔧 Kurulum

1. Projeyi klonlayın:
```bash
git clone https://github.com/nihatxp/links.git
cd links
```

2. Dosyaları bir web sunucusunda çalıştırın:
```bash
# Python ile
python -m http.server 8000

# Node.js ile
npx serve .

# PHP ile
php -S localhost:8000
```

3. Tarayıcınızda `http://localhost:8000` adresini açın.
